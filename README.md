# Explainable-AI-Final-Assignment
Explainable AI assignment using SHAP and counterfactuals

## Installation
Install required packages:
pip install pandas numpy matplotlib scikit-learn shap dice-ml

## How to run
1. Place "dataset.csv" in the same folder as the notebook "xai.ipynb"
2. Open the notebook
3. Run all cells
